# Emlakçı Web Sitesi  
  
## Kullandığım Teknolojiler  
- ASP.Net Core 6.0
- MSSQL
- Dapper
- ASP.Net Web API
- Ajax
- SignalR
- Json Web Token
- Html
- Css
- Javascript
- Bootstrap

## Front End
- Html
- Css
- Javascript
- Bootstrap

## Back End  
- C#
- MSSQL
- Swagger


## Öne Çıkan Özellikler
- Dilediğin evi filtreleme özelliği
- Kolay emlak ilanı
- Emlakçılarla mesajlaşma imkanı
- Json Web Token ile Hesap koruması
- Admin ve Üye Paneli
- Emlak İstatistikleri
- MSSQL İlişkili Tablolar

## Teknik Özellikler
- ASP.Net Core 6.0
- MSSQL Veri Tabanı
- ASP.Net Core Web API
- Dapper
- SignalR
- Ajax
